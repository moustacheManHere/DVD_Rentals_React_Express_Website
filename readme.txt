How to Set Up:

- In your mysql workbench run the "newSakila.sql" file to create a database (delete the old one if it exists)
- Open this project in visualStudio 
- in the terminal make sure you are in the directory that hold both front end and back end
- run "npm start"

- For user, you can test login with "PATRICIA.JOHNSON@sakilacustomer.org" and  "yes"
- For admin you can test login with "admin" and "admin"
